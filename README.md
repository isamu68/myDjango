### このアプリケーションでは、下記の機能を実現する。
1. 管理画面でのマスターデータの管理
2. お客様情報一覧表、検索、フィルター、表示項目の選択、項目ごとのソート機能
3. 入力項目のSuggest機能の追加
4. 入力項目のチェック
5. お客様情報の詳細の表示画面と編集画面
6. お客様情報の新規登録、編集、削除機能
